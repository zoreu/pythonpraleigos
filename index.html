Conceitos basicos de python


Python é uma linguagem de programação de alto nível e interpretada. Isso significa que é uma linguagem de programação fácil de ler e escrever, e que o código Python pode ser executado diretamente sem precisar ser compilado.

Alguns conceitos básicos de Python incluem:
Variáveis: um espaço na memória do seu computador onde você pode armazenar um valor. Por exemplo:

x = 5
y = "Oi, mundo!"

Operadores: símbolos que realizam operações matemáticas ou lógicas em seu código. Alguns exemplos comuns são + (adição), - (subtração), * (multiplicação), / (divisão) e == (igualdade).
Listas: Listas são sequências ordenadas de valores que podem ser de qualquer tipo. Elas são criadas usando colchetes [] e os valores são separados por vírgulas. Por exemplo:

minha_lista = [1, 2, 3, 4]

Você pode acessar os elementos de uma lista usando seus índices (a posição do elemento na lista), começando por 0. Por exemplo:

minha_lista = [1, 2, 3, 4]
print(minha_lista[0]) # imprime 1
print(minha_lista[2]) # imprime 3

Você também pode usar os operadores de slice para obter sublistas de uma lista. esses operadores funcionam assim:

lista[indice_inicio:indice_final] exemplo:

minha_lista = [1, 2, 3, 4, 5]
print(minha_lista[1:3]) # imprime [2, 3]
print(minha_lista[:2]) # imprime [1, 2]
print(minha_lista[2:]) # imprime [3, 4, 5]

Tuplas: Tuplas são muito semelhantes às listas, mas são imutáveis (ou seja, uma vez criadas, não podem ser modificadas). Elas são criadas usando parênteses () e os valores são separados por vírgulas. Por exemplo:

minha_tupla = (1, 2, 3, 4)

Você pode acessar os elementos de uma tupla da mesma maneira que uma lista, usando seus índices e slices.

Dicionários: os dicionários são estruturas de dados que armazenam pares chave-valor. As chaves são únicas em um dicionário e os valores podem ser de qualquer tipo. Os dicionários são criados usando chaves {} e os pares chave-valor são separados por vírgulas. Por exemplo:

meu_dicionario = {'nome': 'João', 'idade': 30}

Você pode acessar os valores em um dicionário usando suas chaves. Por exemplo:

print(meu_dicionario['nome']) # imprime "João"
print(meu_dicionario['idade']) # imprime 30

Usando get para retornar uma exceção:
as vezes pode ser nescessário colocar exceção ao acessar dicionario, pois usando colchete [] pra acessar o dicionario e o mesmo não existe será exibido um erro,
nesse caso pode ser últil colocar o get pra da uma exceção, exemplo:
meu_dicionario = {'nome': 'João', 'idade': 30}
print(meu_dicionario.get('profissao', 'Não existe a chave profissao')) # imprime "Não existe a chave profissao" pois não existe no dicionario

Strings: As strings em Python são sequências de caracteres que podem ser usadas para armazenar e manipular texto. Elas podem ser criadas usando aspas simples ou duplas:

string1 = 'Oi, mundo!'
string2 = "Oi, mundo!"

Concatenando strings:
string1 = 'Oi, '
string2 = 'mundo!'
greeting = string1 + string2
print(greeting) # imprime "Oi, mundo!"

Você também pode usar aspas triplas para criar strings multilinhas:

string3 = """Oi,
mundo!"""

Você também pode usar a função format() de strings para inserir valores de variáveis em uma string. Por exemplo:
name = 'João'
age = 30
string = "Meu nome é {} e tenho {} anos".format(name, age)
print(string) # imprime "Meu nome é João e tenho 30 anos"

Concatenando listas:

Para concatenar (juntar) duas listas em Python, você pode usar o operador +. Por exemplo:

lista1 = [1, 2, 3]
lista2 = [4, 5, 6]
lista_concatenada = lista1 + lista2
print(lista_concatenada) # imprime [1, 2, 3, 4, 5, 6]
Você também pode usar o método extend() de uma lista para adicionar os elementos de outra lista a ela. Por exemplo:

lista1 = [1, 2, 3]
lista2 = [4, 5, 6]
lista1.extend(lista2)
print(lista1) # imprime [1, 2, 3, 4, 5, 6]

Concatenando tuplas:

é possível concatenar tuplas em Python da mesma maneira que listas, usando o operador + Por exemplo:

tupla1 = (1, 2, 3)
tupla2 = (4, 5, 6)
tupla_concatenada = tupla1 + tupla2
print(tupla_concatenada) # imprime (1, 2, 3, 4, 5, 6)

tupla1 = (1, 2, 3)
tupla2 = (4, 5, 6)
tupla1 += tupla2
print(tupla1) # imprime (1, 2, 3, 4, 5, 6)

No entanto, é importante lembrar que as tuplas são imutáveis, o que significa que você não pode modificar os elementos de uma tupla diretamente. Em vez disso, você precisará criar uma nova tupla com os elementos desejados.

Laço de repetição 'for':

O laço 'for' em Python é usado para iterar sobre um iterável, como uma lista ou uma string. Aqui está um exemplo de como usar o 'for' para percorrer uma lista:

numbers = [1, 2, 3, 4, 5]
for number in numbers:
    print(number)
Neste exemplo, a variável number tomará cada valor na lista numbers sucessivamente, e o código dentro do laço será executado para cada valor. O resultado seria a impressão de cada número na lista:

1
2
3
4
5

Você também pode usar o for para iterar sobre uma string, como neste exemplo:

word = "hello"
for letter in word:
    print(letter)
Isso imprimiria cada letra da palavra "hello" em uma linha separada:

h
e
l
l
o

Você também pode usar o for em combinação com a função range() para iterar um número específico de vezes. Por exemplo:
for i in range(5):
    print(i)
Isso imprimiria os números de 0 a 4 (lembre-se que o Python começa a contar do 0, não do 1).

você pode desempacotar uma tupla em python atribuindo variaveis a cada elemento da tupla

t = (1,2,3)
n1,n2,n3 = t

pode ser interessante trocar valores de variaveis no python, exemplo:

variavel1 = 'João'
variavel2 = 'Pedro'
variavel3 = 'Rafael'
variavel4 = 'Suzana'
variavel1,variavel2 = variavel3,variavel4
nesse exemplo usamos apenas uma linha pra trocar as variaveis, a variavel1 será variavel3 e a variavel3 será variavel1, a variavel2 será variavel4 e a variavel4 será variavel2, nunca usamos varias linhas porque isso apenas referencia outra variavel e não troca de verdade a variavel

Condições:

if x < y:

Neste exemplo, apenas a última comparação será verdadeira, então apenas a mensagem "x é diferente de y" será impressa.

Você também pode usar operadores lógicos, como and, or e not, para combinar múltiplas condições em uma única expressão. Por exemplo:

x = 10

if x < y and x % 2 == 0:
    print("x é menor que y e é um número par")

if x < y or x % 2 == 0:
    print("x é menor que y ou é um número par")

if not (x < y):
    print("x não é menor que y")

startswith() e endswith()

As funções startswith() e endswith() em Python são métodos de string que verificam se uma string começa ou termina, respectivamente, com uma determinada sequência de caracteres. Aqui está um exemplo de como usá-las:

word = "hello"
if word.startswith("h"):
    print("A string começa com 'h'")

if word.endswith("o"):
    print("A string termina com 'o'")
Isso imprimiria as duas mensagens, pois a string "hello" começa com "h" e termina com "o". Você também pode passar uma tupla como argumento para esses métodos, para verificar se a string começa ou termina com qualquer uma das sequências de caracteres na tupla. Por exemplo:

word = "hello"
if word.startswith(("h", "he", "hel")):
    print("A string começa com 'h', 'he' ou 'hel'")

if word.endswith(("o", "lo", "llo")):
    print("A string termina com 'o', 'lo' ou 'llo'")
Esses métodos também suportam o argumento opcional start e end, que permitem especificar a posição na string a partir da qual a verificação deve começar ou terminar, respectivamente. Por exemplo:

word = "hello"
if word.startswith("e", 1, 3):
    print("A string começa com 'e' entre as posições 1 e 3")

if word.endswith("l", 2):
    print("A string termina com 'l' a partir da posição 2")

Laço de repetição "while":

O laço while em Python é usado para executar um bloco de código repetidamente enquanto uma determinada condição é verdadeira. Aqui está um exemplo de como usar o while:

i = 0
while i < 5:
    print(i)
    i += 1
Neste exemplo, o laço while continuará executando enquanto i for menor que 5. O código dentro do laço imprime o valor de i e, em seguida, adiciona 1 a ele. Isso resultará na impressão dos números de 0 a 4.

Para parar um laço while, você pode usar a palavra-chave break. Isso sairá do laço imediatamente, independentemente da condição de parada ainda ser verdadeira ou não. Por exemplo:

i = 0
while True:
    print(i)
    i += 1
    if i >= 5:
        break

Neste exemplo, o laço while irá iterar cinco vezes, mas apenas os números ímpares serão impressos: 1, 3 e 5. Quando i é um número par, a instrução continue é atingida e o código restante no laço é pulado, fazendo com que o laço volte para o início da próxima iteração.

Conceitos Avançados


Funções:

Em Python, uma função é um bloco de código que pode ser executado repetidamente em diferentes partes do seu programa. Você pode criar uma função para realizar uma tarefa específica ou para calcular um resultado e retorná-lo para ser usado em outro lugar.
Por exemplo, você pode criar uma função para calcular o fatorial de um número, ou para verificar se um número é primo. Depois de criar a função, você pode chamá-la quantas vezes quiser e passar os argumentos necessários como entrada.

A sintaxe para definir uma função em Python é:

def nome_da_funcao(parametros):
    # código da função
    return resultado

Por exemplo, aqui está uma função que calcula o fatorial de um número:

def fatorial(n):
    if n == 0:
        return 1
    else:
        return n * fatorial(n-1)
fatorial_de_5 = fatorial(5)
print(fatorial_de_5) # exibe 120

outros exemplos:
def minha_funcao(parametro1, parametro2):
    # código da função
    resultado = parametro1 + parametro2
    return resultado

resultado = minha_funcao(1, 2)
print(resultado) # exibe 3
Em Python, é possível definir um valor padrão para um ou mais dos parâmetros de uma função. Isso significa que, se o argumento correspondente não for fornecido quando a função for chamada, o valor padrão será usado.

Para definir um valor padrão para um parâmetro, basta atribuir o valor desejado a ele na definição da função, como mostrado abaixo:

def minha_funcao(parametro1, parametro2=valor_padrao):
    # código da função

Por exemplo, aqui está uma função que calcula o fatorial de um número, com um valor padrão de 1 para o parâmetro:

def fatorial(n=1):
    resultado = 1
    for i in range(1, n+1):
        resultado *= i
    return resultado

fatorial_de_5 = fatorial(5)
print(fatorial_de_5) # exibe 120

fatorial_de_1 = fatorial()
print(fatorial_de_1) # exibe 1

No exemplo acima, a função fatorial() tem um parâmetro opcional n, que tem um valor padrão de 1. Se você chamar a função sem passar nenhum argumento, o valor padrão será usado e o resultado será 1. Se você passar um argumento, o valor padrão será sobrescrito e o cálculo será realizado com o novo valor.

Retornando varios valores de uma função:

Para retornar vários valores de uma função, basta colocá-los separados por vírgulas, e então usar a palavra-chave return para retorná-los. Por exemplo:

def minha_funcao():
    valor1 = 1
    valor2 = 2
    valor3 = 3
    return valor1, valor2, valor3

resultado = minha_funcao()
print(resultado) # exibe (1, 2, 3)

No exemplo acima, a função minha_funcao() retorna três valores: valor1, valor2 e valor3. Esses valores são retornados como uma tupla, que pode ser armazenada em uma variável ou usada diretamente.

Você também pode desempacotar a tupla retornada diretamente em várias variáveis, como mostrado abaixo:

valor1, valor2, valor3 = minha_funcao()
print(valor1) # exibe 1
print(valor2) # exibe 2
print(valor3) # exibe 3

Decoradores:

Os decoradores são uma forma de aplicar modificações a funções existentes de forma transparente, sem precisar modificar o código da função original. Eles são usados para adicionar funcionalidades adicionais às funções, como registrar informações de depuração, medir o tempo de execução, validar entradas, etc.

Para criar um decorador, você precisa definir uma função que recebe uma outra função como argumento e retorna uma nova função que envolve a função original. A sintaxe para criar um decorador é:

def decorador(funcao):
    def wrapper(*args, **kwargs):
        # código adicional
        resultado = funcao(*args, **kwargs)
        # mais código adicional
        return resultado
    return wrapper

O decorador acima tem uma função interna chamada wrapper, que é responsável por realizar as modificações na função original. A função wrapper pode ser personalizada para adicionar qualquer tipo de funcionalidade adicional que você desejar.

Para usar o decorador, basta colocar o símbolo @ seguido pelo nome do decorador acima da definição da função que deseja decorar. Por exemplo:

@decorador
def minha_funcao(parametro):
    # código da função


Quando a função é chamada, o decorador é aplicado automaticamente e as modificações definidas na função wrapper são realizadas.

Aqui está um exemplo de decorador que mede o tempo de execução de uma função:

import time
def tempo_de_execucao(funcao):
    def wrapper(*args, **kwargs):
        inicio = time.time()
        resultado = funcao(*args, **kwargs)
        fim = time.time()
        tempo = fim - inicio
        print(f'Tempo de execução: {tempo:.6f} segundos')
        return resultado
    return wrapper

@tempo_de_execucao
def minha_funcao(parametro):
    # código da função
minha_funcao(12345) # exibe o tempo de execução da função

Em Python, é possível passar argumentos para um decorador da mesma forma que passa argumentos para qualquer outra função. Para fazer isso, basta colocar os argumentos entre parênteses logo após o nome do decorador, quando o decorador é aplicado a uma função.

Por exemplo, aqui está um decorador que recebe um argumento nome e exibe esse nome junto com o resultado da função decorada:

def meu_decorador(nome):
    def decorador(funcao):
        def wrapper(*args, **kwargs):
            resultado = funcao(*args, **kwargs)
            print(f'{nome}: {resultado}')
            return resultado
        return wrapper
    return decorador

@meu_decorador('Resultado')
def minha_funcao(parametro):
    # código da função
    return parametro * 2

minha_funcao(10) # exibe 'Resultado: 20'

Orientação a objetos:

O Python é uma linguagem de programação orientada a objetos, o que significa que ela permite a criação e uso de objetos para representar coisas do mundo real.

Em Python, um objeto é uma instância de uma classe. Uma classe é um modelo ou protótipo para criar objetos. Ela define os atributos e comportamentos que um objeto terá.

Para criar uma classe em Python, você deve usar a palavra-chave class, seguida pelo nome da classe e dois-pontos. Em seguida, você deve definir o método __init__(), que é o construtor da classe. Ele é chamado quando um objeto é criado e permite inicializar os atributos do objeto.

Aqui está um exemplo de como criar uma classe em Python:

class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade


Para criar um objeto a partir da classe Pessoa, basta usar o nome da classe como uma função e passar os argumentos para o construtor __init__():

pessoa1 = Pessoa("João", 30)
pessoa2 = Pessoa("Maria", 25)

Os atributos nome e idade podem ser acessados diretamente usando a notação de ponto:

print(pessoa1.nome) # exibe "João"
print(pessoa2.idade) # exibe 25

Além de atributos, uma classe também pode ter métodos, que são funções que operam sobre os atributos do objeto. Para criar um método, basta definir uma função dentro da classe, usando a palavra-chave >def.

Aqui está um exemplo de como adicionar um método à classe Pessoa:

class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade

    def aniversario(self):
        self.idade += 1

Para chamar um método de um objeto, basta usar a notação de ponto e passar qualquer argumento necessário:

pessoa1.aniversario()
print(pessoa1.idade) # exibe 31

Passando argumentos para o método da classe:

Para passar argumentos para um método de um objeto em Python, basta chamar o método com a notação de ponto e passar os argumentos entre parênteses, da mesma forma que faria com uma função normal.

Por exemplo, suponha que você tenha a seguinte classe Pessoa:

class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade

    def cumprimentar(self, nome_pessoa):
        print(f"Olá, {nome_pessoa}, meu nome é {self.nome}")

Para chamar o método cumprimentar() com um argumento, basta fazer:

pessoa1 = Pessoa("João", 30)
pessoa1.cumprimentar("Maria") # exibe "Olá, Maria, meu nome é João"

O primeiro argumento passado para o método é sempre o próprio objeto, que é referenciado como self. Qualquer outro argumento é passado normalmente.

Retornando variaveis de um objeto

Para retornar um atributo de um objeto em Python, basta acessá-lo diretamente usando a notação de ponto. Por exemplo:

class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade

    def get_nome(self):
        return self.nome

pessoa1 = Pessoa("João", 30)
nome = pessoa1.get_nome()
print(nome) # exibe "João"

Usando o decorador @classmethod

O decorador @classmethod em Python é usado para definir um método da classe, ou seja, um método que é acessado diretamente a partir da classe, e não a partir de uma instância da classe (objeto). Um método da classe é chamado da mesma forma que um atributo da classe, usando a notação de ponto. Ele recebe como primeiro argumento a própria classe, que é referenciada como cls.

Por exemplo, suponha que você tenha a seguinte classe Pessoa:

class Pessoa:
    num_pessoas = 0

    def __init__(self, nome):
        self.nome = nome
        Pessoa.num_pessoas += 1

    @classmethod
    def get_num_pessoas(cls):
        return cls.num_pessoas

A classe Pessoa possui um atributo da classe num_pessoas, que é inicializado com 0, e um método da classe get_num_pessoas(), que retorna o valor de num_pessoas. Para acessar o método da classe, basta usar a notação de ponto:

pessoa1 = Pessoa("João")
pessoa2 = Pessoa("Maria")
num_pessoas = Pessoa.get_num_pessoas()
print(num_pessoas) # exibe 2

No exemplo acima, o método get_num_pessoas() é chamado diretamente a partir da classe Pessoa, sem precisar de uma instância da classe (objeto). O primeiro argumento passado para o método é a própria classe, que é referenciada como cls.

Usando o decorador @staticmethod

O decorador @staticmethod em Python é usado para definir um método estático, ou seja, um método que não recebe nenhum argumento especial, nem a classe nem o objeto.

Um método estático é chamado da mesma forma que um atributo da classe, usando a notação de ponto. Ele não recebe nenhum argumento especial e pode ser chamado tanto a partir da classe quanto a partir de uma instância da classe (objeto).

Por exemplo, suponha que você tenha a seguinte classe Pessoa:

class Pessoa:
    @staticmethod
    def saludar():
        print("Hola")

A classe Pessoa possui um método estático saludar(), que imprime "Hola".

Para chamar o método estático, basta usar a notação de ponto:

Pessoa.saludar() # exibe "Hola"

pessoa1 = Pessoa()
pessoa1.saludar() # exibe "Hola"

No exemplo acima, o método saludar() é chamado tanto diretamente a partir da classe Pessoa, quanto a partir de uma instância da classe (objeto). Como é um método estático, ele não recebe nenhum argumento especial.

Usando método find()

O método find() é um método de string que procura por uma substring em uma string e retorna o índice em que a substring foi encontrada. Se a substring não for encontrada, o método retorna -1.

A sintaxe para usar o método find() é:

string.find(substring, start, end)

  • substring é a substring que deseja procurar.
  • start é o índice de início da busca (opcional).
  • end é o índice de fim da busca (opcional).


Se você não especificar os índices de início e fim, o método irá procurar a substring em toda a string. Se você especificar apenas o índice de início, o método irá procurar a substring a partir do índice especificado até o final da string.

Aqui estão alguns exemplos de como usar o método find():

texto = "O rato roeu a roupa do rei de Roma"

# procura a substring "rei" em toda a string
indice = texto.find("rei")
print(indice) # exibe 24
# procura a substring "Roma" a partir do índice 30
indice = texto.find("Roma", 30)
print(indice) # exibe 37
# procura a substring "raposa" em toda a string
indice = texto.find("raposa")
print(indice) # exibe -1

A sintaxe para usar o método len() é:

len(objeto)

Onde objeto é o objeto cujo tamanho deseja verificar.

Aqui estão alguns exemplos de como usar o método len():

# verifica o tamanho de uma string
texto = "O rato roeu a roupa do rei de Roma"

dicionario = {'chave1': 1, 'chave2': 2, 'chave3': 3}

Usando split()

O método split() é um método de string que divide uma string em uma lista de substrings, usando um delimitador especificado. O delimitador é um caractere ou sequência de caracteres que separa as substrings na string original.

A sintaxe para usar o método split() é:

string.split(delimitador, maxsplit)
  • delimitador é o caractere ou sequência de caracteres que serve como delimitador. Se você não especificar um delimitador, o método usará o espaço em branco como delimitador.
  • maxsplit é o número máximo de divisões a serem feitas (opcional). Se você não especificar um valor para maxsplit, todas as ocorrências do delimitador serão usadas.


Aqui estão alguns exemplos de como usar o método split():

# divide uma string usando o espaço em branco como delimitador
texto = "O rato roeu a roupa do rei de Roma"
palavras = texto.split()
print(palavras) # exibe ['O', 'rato', 'roeu', 'a', 'roupa', 'do', 'rei', 'de', 'Roma']

# divide uma string usando vírgulas como delimitador
texto = "O rato,roeu,a roupa,do rei de Roma"
palavras = texto.split(',')
print(palavras) # exibe ['O rato', 'roeu', 'a roupa', 'do rei de Roma']

# divide uma string em duas partes, usando o primeiro espaço em branco como delimitador
texto = "O rato roeu a roupa do rei de Roma"
partes = texto.split(None, 1)
print(partes) # exibe ['O', 'rato roeu a roupa do rei de Roma']

# divide uma string usando vírgulas como delimitador, mas apenas as duas primeiras ocorrências
texto = "O rato,roeu,a roupa,do rei de Roma"
palavras = texto.split(',', 2)
print(palavras) # exibe ['O rato', 'roeu', 'a roupa,do rei de Roma']

Codificando em UTF-8

O Python possui o módulo codecs que fornece as funções encode() e decode() para codificar e decodificar strings em diferentes codificações de caracteres, como UTF-8, ASCII, etc.

A sintaxe para codificar uma string em UTF-8 é:

import codecs
string_codificada = string.encode('utf-8')

Onde string é a string que deseja codificar. A função encode() retornará a string codificada como um objeto bytes.

Para decodificar a string, basta usar a função decode() da seguinte forma:

import codecs
string_decodificada = string_codificada.decode('utf-8')

Onde string_codificada é o objeto bytes com a string codificada em UTF-8. A função decode() retornará a string decodificada como uma string normal.

Exemplo completo:

import codecs
# codifica a string em UTF-8
string = "O rato roeu a roupa do rei de Roma"
string_codificada = string.encode('utf-8')
print(string_codificada) # exibe b'O rato roeu a roupa do rei de Roma'

# decodifica a string
string_decodificada = string_codificada.decode('utf-8')
print(string_decodificada) # exibe 'O rato roeu a roupa do rei de Roma'

Codificando em Base64

Base64 é um sistema de codificação de dados que permite representar qualquer conjunto de dados binários como uma sequência de caracteres. É amplamente usado para transmitir dados binários através de canais que só podem transmitir dados ASCII, como e-mails, mensagens de texto e outras formas de comunicação.

O Python possui o módulo base64 que fornece as funções b64encode() e b64decode() para codificar e decodificar dados base64, respectivamente.

A sintaxe para codificar dados base64 é:

import base64
dados_codificados = base64.b64encode(dados)

Onde dados é o objeto bytes com os dados a serem codificados. A função b64encode() retornará os dados codificados como um objeto bytes.

Para decodificar os dados, basta usar a função b64decode() da seguinte forma:

import base64
dados_decodificados = base64.b64decode(dados_codificados)

Onde dados_codificados é o objeto bytes com os dados base64 codificados. A função b64decode() retornará os dados decodificados como um objeto bytes.

Aqui está um exemplo completo de como codificar e decodificar dados base64 em Python:

import codecs
import base64
# codifica os dados
dados = 'O rato roeu a roupa do rei de Roma'.encode('utf-8')

print(dados_codificados) # exibe TyByYXRvIHJvZXUgYSByb3VwYSBkbyByZWkgZGUgUm9tYQ==

import base64

Expressões regulares:

As expressões regulares (ou regex) são um conjunto de caracteres que formam um padrão de busca em uma string. Elas são usadas para verificar se uma string possui o padrão especificado e podem ser usadas para substituir ou extrair partes de uma string.

O Python possui o módulo re (para "regular expression") que fornece diversas funções para trabalhar com expressões regulares.

import re

A função mais básica do módulo re é a search(), que verifica se uma string possui o padrão especificado. Ela retorna um objeto Match se o padrão for encontrado ou None se o padrão não for encontrado. A sintaxe para usar a função search() é:

resultado = re.search(padrao, string)

Onde padrao é a expressão regular a ser procurada e string é a string na qual a expressão regular será procurada.

Por exemplo, suponha que você queira verificar se a string "O rato roeu a roupa do rei de Roma" possui a palavra "roupa":

import re

string = "O rato roeu a roupa do rei de Roma"
padrao = "roupa"
resultado = re.search(padrao, string)

if resultado:
    print("Padrão encontrado!")
else:
    print("Padrão não encontrado")

O código acima imprime "Padrão encontrado!", pois a string "roupa" foi encontrada na string de entrada.

Além da função search(), o módulo re possui diversas outras funções úteis para trabalhar com expressões regulares, como findall(), que encontra todas as ocorrências de um padrão em uma string; sub(), que substitui todas as ocorrências de um padrão em uma string; e split(), que divide uma string em uma lista de strings usando um padrão como delimitador.

Usando padrões no re

Existem vários caracteres especiais que podem ser usados em expressões regulares para criar padrões mais complexos. Alguns exemplos de caracteres especiais comuns são:

  • "." (ponto): significa "qualquer caractere", exceto uma nova linha. Por exemplo, o padrão "a.c" casaria com "abc", "a1c", "a#c", etc.
  • "^" (circunflexo): significa "começo da linha". Por exemplo, o padrão "^O" casaria com "O rato roeu a roupa do rei de Roma", mas não com "E o rato roeu a roupa do rei de Roma".
  • "$" (cifrão): significa "fim da linha". Por exemplo, o padrão "Roma$" casaria com "O rato roeu a roupa do rei de Roma", mas não com "O rato roeu a roupa do rei de Roma.".
  • "*" (asterisco): significa "zero ou mais ocorrências"
  • "+" (mais): significa "uma ou mais ocorrências". Por exemplo, o padrão "a+b" casaria com "ab", "aab", "aaab", etc., mas não com "b".
  • "?" (interrogação): significa "zero ou uma ocorrência". Por exemplo, o padrão "a?b" casaria com "b" e "ab", mas não com "aab" ou "aaab".
  • "^[a-z]+$" qualquer string que consista apenas em letras minúsculas e tenha pelo menos uma letra
  • "^[0-9]+$" qualquer string que consista apenas em dígitos e tenha pelo menos um dígito.
  • "^[!@#$%^&*()_+-=[]{}|;':",./<>?].$" casaria com qualquer string que contenha pelo menos um caractere especial


Exemplos com modulo re

Por exemplo, o padrão "^[a-z]+$" casaria com qualquer string que consista apenas em letras minúsculas e tenha pelo menos uma letra. Por exemplo:

import re

padrao = "^[a-z]+$"

resultado = re.search(padrao, "abcdef")
print(resultado) # exibe <re.Match object; span=(0, 6), match='abcdef'>

resultado = re.search(padrao, "ABCDEF")
print(resultado) # exibe None

resultado = re.search(padrao, "abc123")
print(resultado) # exibe None

No exemplo acima, o padrão "^[a-z]+$" casa com a string "abcdef", mas não com as strings "ABCDEF" ou "abc123".

Por exemplo, o padrão "^[0-9]+$" casaria com qualquer string que consista apenas em dígitos e tenha pelo menos um dígito. Por exemplo:

import re

padrao = "^[0-9]+$"

resultado = re.search(padrao, "12345")
print(resultado) # exibe <re.Match object; span=(0, 5), match='12345'>
resultado = re.search(padrao, "ABCDEF")
print(resultado) # exibe None

resultado = re.search(padrao, "abc123")
print(resultado) # exibe None

No exemplo acima, o padrão "^[0-9]+$" casa com a string "12345", mas não com as strings "ABCDEF" ou "abc123".

Por exemplo, o padrão "^[!@#$%^&*()_+-=[]{}|;':",./<>?].+$" casaria com qualquer string que contenha pelo menos um caractere especial. Por exemplo:

import re

padrao = "^[!@#$%^&*()_+-=[]{}|;':",./<>?].+$"

resultado = re.search(padrao, "abc123!@#$")
print(resultado) # exibe <re.Match object; span=(0, 9), match='abc123!@#$'>

resultado = re.search(padrao, "ABCDEF")
print(resultado) # exibe None

resultado = re.search(padrao, "abc123")
print(resultado) # exibe None

No exemplo acima, o padrão "^[!@#$%^&*()_+-=[]{}|;':",./<>?].+$" casa com a string "abc123!@#$", mas não com as strings "ABCDEF" ou "abc123".
usando findall() do re

Para usar a função findall() do módulo re, basta chamá-la passando como argumentos a expressão regular a ser procurada e a string na qual a expressão regular será procurada. A função findall() retorna uma lista com todas as ocorrências encontradas.

Por exemplo, suponha que você queira encontrar todas as ocorrências de uma palavra que comece com "r" e tenha pelo menos um caractere depois na string "O rato roeu a roupa do rei de Roma":

import re

string = "O rato roeu a roupa do rei de Roma"
padrao = "r.+"
resultado = re.findall(padrao, string)
print(resultado) # exibe ['rato', 'roeu', 'roupa', 'rei', 'roma']

usando sub() do re

Para usar a função sub() do módulo re, basta chamá-la passando como argumentos a expressão regular a ser procurada, a string de substituição e a string na qual a expressão regular será procurada. A função sub() retorna uma nova string com as ocorrências da expressão regular substituídas pela string de substituição.

Por exemplo, suponha que você queira substituir todas as ocorrências da palavra "roupa" por "calça" na string "O rato roeu a roupa do rei de Roma":

import re

padrao = "roupa"
substituicao = "calça"
resultado = re.sub(padrao, substituicao, string)
print(resultado) # exibe "O rato roeu a calça do rei de Roma"

No exemplo acima, a função sub() retorna a string "O rato roeu a calça do rei de Roma", pois todas as ocorrências da palavra "roupa" foram substituídas pela palavra "calça".

Usando o método replace()

O método replace() é um método de strings que permite substituir uma ou mais ocorrências de uma substring por outra substring.

Para usar o método replace(), basta chamar o método na string em que você quer realizar a substituição, passando como argumentos a substring a ser substituída e a substring de substituição. O método replace() retorna uma nova string com as substituições realizadas.

Por exemplo, suponha que você queira substituir todas as ocorrências da palavra "roupa" por "calça" na string "O rato roeu a roupa do rei de Roma":

string = "O rato roeu a roupa do rei de Roma"
resultado = string.replace("roupa", "calça")
print(resultado) # exibe "O rato roeu a calça do rei de Roma"

No exemplo acima, o método replace() retorna a string "O rato roeu a calça do rei de Roma", pois todas as ocorrências da palavra "roupa" foram substituídas pela palavra "calça".

Imports



Em Python, os imports permitem que você use módulos ou funções de outros arquivos em seu código.

Para importar um módulo ou função em Python, basta usar a palavra-chave import seguida do nome do módulo ou função. Por exemplo, para importar o módulo math, basta usar a seguinte instrução:

import math

Depois de importar o módulo, você pode usar qualquer função ou variável definida no módulo usando o nome do módulo como prefixo. Por exemplo, para usar a função sqrt() do módulo math, basta usar o seguinte código:

import math

x = math.sqrt(9)

from math import sqrt

x = sqrt(9)
print(x) # exibe 3.0

import requests

url = 'https://www.example.com/login'

    print('Success')

    print('Failed')

Isso enviará uma requisição POST para a URL https://www.example.com/login com os dados de login especificados no dicionário data.

Esse código envia uma requisição POST para a URL https://www.example.com/upload com o arquivo myfile.txt anexado.

obter o valor de um atributo de uma tag HTML

Por exemplo, suponha que você tenha o seguinte HTML:

<div data-texto="texto-1">Texto da div 1</div>
<div data-texto="texto-2">Texto da div 2</div>
<div data-texto="texto-3">Texto da div 3</div>

Para obter o valor do atributo data-texto da primeira div, basta usar o seguinte código:

from bs4 import BeautifulSoup

html = '<div data-texto="texto-1">Texto da div 1</div><div data-texto="texto-2">Texto da div 2</div><div data-texto="texto-3">Texto da div 3</div>'
soup = BeautifulSoup(html, 'html.parser')
div = soup.find('div')
valor_atributo = div.get('data-texto')
print(valor_atributo) # imprime "texto-1"
