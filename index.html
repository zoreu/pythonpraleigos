Como instalar Python


Para instalar o python e começar a desenvolver nessa linguagem de programação, acesse: https://www.python.org/downloads/
Escolha a versão mais recente do Python (atualmente, a versão 3.10.0) e siga as instruções de instalação. É geralmente uma boa ideia aceitar as opções padrão durante a instalação.

Conceitos basicos de python


Python é uma linguagem de programação de alto nível e interpretada. Isso significa que é uma linguagem de programação fácil de ler e escrever, e que o código Python pode ser executado diretamente sem precisar ser compilado.

Alguns conceitos básicos de Python incluem:
Variáveis: um espaço na memória do seu computador onde você pode armazenar um valor. Por exemplo:

x = 5
y = "Oi, mundo!"

Operadores: símbolos que realizam operações matemáticas ou lógicas em seu código. Alguns exemplos comuns são + (adição), - (subtração), * (multiplicação), / (divisão) e == (igualdade).
Listas: Listas são sequências ordenadas de valores que podem ser de qualquer tipo. Elas são criadas usando colchetes [] e os valores são separados por vírgulas. Por exemplo:

minha_lista = [1, 2, 3, 4]

Você pode acessar os elementos de uma lista usando seus índices (a posição do elemento na lista), começando por 0. Por exemplo:

minha_lista = [1, 2, 3, 4]
print(minha_lista[0]) # imprime 1
print(minha_lista[2]) # imprime 3

Você também pode usar os operadores de slice para obter sublistas de uma lista. esses operadores funcionam assim:

lista[indice_inicio:indice_final] exemplo:

minha_lista = [1, 2, 3, 4, 5]
print(minha_lista[1:3]) # imprime [2, 3]
print(minha_lista[:2]) # imprime [1, 2]
print(minha_lista[2:]) # imprime [3, 4, 5]

Tuplas: Tuplas são muito semelhantes às listas, mas são imutáveis (ou seja, uma vez criadas, não podem ser modificadas). Elas são criadas usando parênteses () e os valores são separados por vírgulas. Por exemplo:
minha_tupla = (1, 2, 3, 4)
Você pode acessar os elementos de uma tupla da mesma maneira que uma lista, usando seus índices e slices.

Dicionários: os dicionários são estruturas de dados que armazenam pares chave-valor. As chaves são únicas em um dicionário e os valores podem ser de qualquer tipo. Os dicionários são criados usando chaves {} e os pares chave-valor são separados por vírgulas. Por exemplo:

meu_dicionario = {'nome': 'João', 'idade': 30}

Você pode acessar os valores em um dicionário usando suas chaves. Por exemplo:

print(meu_dicionario['nome']) # imprime "João"
print(meu_dicionario['idade']) # imprime 30

Usando get para retornar uma exceção:
as vezes pode ser nescessário colocar exceção ao acessar dicionario, pois usando colchete [] pra acessar o dicionario e o mesmo não existe será exibido um erro,
nesse caso pode ser últil colocar o get pra da uma exceção, exemplo:
meu_dicionario = {'nome': 'João', 'idade': 30}
print(meu_dicionario.get('profissao', 'Não existe a chave profissao')) # imprime "Não existe a chave profissao" pois não existe no dicionario

Strings: As strings em Python são sequências de caracteres que podem ser usadas para armazenar e manipular texto. Elas podem ser criadas usando aspas simples ou duplas:

string1 = 'Oi, mundo!'
string2 = "Oi, mundo!"

Concatenando strings:
string1 = 'Oi, '
string2 = 'mundo!'
greeting = string1 + string2
print(greeting) # imprime "Oi, mundo!"

Você também pode usar aspas triplas para criar strings multilinhas:

string3 = """Oi,
mundo!"""

Você também pode usar a função format() de strings para inserir valores de variáveis em uma string. Por exemplo:
name = 'João'
age = 30
string = "Meu nome é {} e tenho {} anos".format(name, age)
print(string) # imprime "Meu nome é João e tenho 30 anos"

Concatenando listas:

Para concatenar (juntar) duas listas em Python, você pode usar o operador +. Por exemplo:

lista1 = [1, 2, 3]
lista2 = [4, 5, 6]
lista_concatenada = lista1 + lista2
print(lista_concatenada) # imprime [1, 2, 3, 4, 5, 6]
Você também pode usar o método extend() de uma lista para adicionar os elementos de outra lista a ela. Por exemplo:

lista1 = [1, 2, 3]
lista2 = [4, 5, 6]
lista1.extend(lista2)
print(lista1) # imprime [1, 2, 3, 4, 5, 6]

Concatenando tuplas:

é possível concatenar tuplas em Python da mesma maneira que listas, usando o operador + Por exemplo:

tupla1 = (1, 2, 3)
tupla2 = (4, 5, 6)
tupla_concatenada = tupla1 + tupla2
print(tupla_concatenada) # imprime (1, 2, 3, 4, 5, 6)

tupla1 = (1, 2, 3)
tupla2 = (4, 5, 6)
tupla1 += tupla2
print(tupla1) # imprime (1, 2, 3, 4, 5, 6)

No entanto, é importante lembrar que as tuplas são imutáveis, o que significa que você não pode modificar os elementos de uma tupla diretamente. Em vez disso, você precisará criar uma nova tupla com os elementos desejados.

Laço de repetição 'for':

O laço 'for' em Python é usado para iterar sobre um iterável, como uma lista ou uma string. Aqui está um exemplo de como usar o 'for' para percorrer uma lista:

numbers = [1, 2, 3, 4, 5]
for number in numbers:
    print(number)
Neste exemplo, a variável number tomará cada valor na lista numbers sucessivamente, e o código dentro do laço será executado para cada valor. O resultado seria a impressão de cada número na lista:

1
2
3
4
5

Você também pode usar o for para iterar sobre uma string, como neste exemplo:

word = "hello"
for letter in word:
    print(letter)
Isso imprimiria cada letra da palavra "hello" em uma linha separada:

h
e
l
l
o

Você também pode usar o for em combinação com a função range() para iterar um número específico de vezes. Por exemplo:
for i in range(5):
    print(i)
Isso imprimiria os números de 0 a 4 (lembre-se que o Python começa a contar do 0, não do 1).

Descompactando tuplas:
você pode descompactar uma tupla em python atribuindo variaveis a cada elemento da tupla

t = (1,2,3)
n1,n2,n3 = t
print(n1)
print(n2)
print(n3)

Trocando variaveis:
pode ser interessante trocar valores de variaveis no python, exemplo:

variavel1 = 'João'
variavel2 = 'Pedro'
variavel3 = 'Rafael'
variavel4 = 'Suzana'
variavel1,variavel2 = variavel3,variavel4
nesse exemplo usamos apenas uma linha pra trocar as variaveis, a variavel1 será variavel3 e a variavel3 será variavel1, a variavel2 será variavel4 e a variavel4 será variavel2, nunca usamos varias linhas porque isso apenas referencia outra variavel e não troca de verdade a variavel

Identação

Dada uma função como 'for' por exemplo, a proxima linha tem q ser identada pois dara erro se deixar centralizada com a primeira linha, exemplo:
Neste exemplo, apertamos a tecla tab uma vez ou apertamos a tecla espaço quatro vezes até ficar identada e poder escrever o proximo codigo:
for letter in word:
    print(letter) # linha identada

no python trabalhamos com identação porque o interpretador do python precisa saber que a proxima linha decorre de uma função anterior, pois o python executa de cima para baixo o codigo.
